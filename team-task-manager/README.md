# Task Management Dashboard

A modern collaborative task management dashboard built using React.js and Firebase.

## Features

- User Authentication
- Task Creation
- Task Editing
- Task Completion
- Task Deletion
- Search Tasks
- Filter Tasks
- Sort Tasks
- Notifications System
- Realtime Team Chat
- Reports & Analytics
- Pie Charts & Bar Charts
- Responsive Dashboard UI

## Technologies Used

- React.js
- Firebase Authentication
- Firebase Firestore
- Tailwind CSS
- Recharts
- React Router DOM

## Installation

1. Clone repository

```bash
git clone YOUR_GITHUB_LINK
```

2. Install dependencies

```bash
npm install
```

3. Start application

```bash
npm run dev
```
